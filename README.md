A responsive Dictionary Web App built using React, TypeScript, and Tailwind CSS.  
It uses the Merriam-Webster API to fetch real-time word meanings, pronunciations, and definitions.

---

## Features

- Search any English word
- Get accurate definitions
- Pronunciation support
- Press Enter to search
- Fast API response
- Responsive design for mobile and desktop

---

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Merriam-Webster Dictionary API
- Vite
